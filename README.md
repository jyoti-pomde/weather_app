Weather App (Flask + OpenWeather API)
📌 Overview
This is a simple web-based Weather Application built using Python Flask.
It fetches real-time weather data using the OpenWeather API and displays temperature, humidity, and weather conditions for any city.

🚀 Features
🔍 Search weather by city name
🌡 Real-time temperature display (°C)
💧 Humidity information
🌤 Weather condition description
⚡ Simple and responsive UI
❌ Error handling for invalid cities


🛠 Tech Stack
Python 🐍
Flask 🌐
HTML5
CSS3
OpenWeather API 🌍
Requests Library
